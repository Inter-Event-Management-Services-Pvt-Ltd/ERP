# MVP Delivery Plan

## Phase 0 — Foundation

- Create Supabase project
- Apply migrations
- Apply seed data
- Configure Google login
- Add approved employees
- Verify Director bootstrap
- Create FastAPI and Next.js repositories
- Configure staging

## Phase 1 — Document and Archive Core

- Client CRUD
- Project CRUD
- Project team assignments
- Folder template application
- Folder tree UI
- Document uploads
- Immutable document versions
- Search
- ZIP archive generation
- Manifest generation
- PDF labels and QR codes
- Physical storage locations
- Physical-file checkout and return

## Phase 2 — Employee Operations

- Attendance check-in and check-out
- Attendance history
- Leave requests
- Tasks
- Task comments and attachments
- Calendar

## Phase 3 — Management Controls

- Approvals
- Director Dashboard
- Audit explorer
- Alerts
- Archive verification
- Admin role management
- ABAC policy UI

## Phase 4 — Improvements

- Mobile scanning
- Preview generation
- Duplicate detection
- Better offline caching
- Desktop sync agent only if proven necessary
- Google Calendar integration
- Payroll export only if needed
