# Phase 5 Local Performance Baseline

Date: 2026-06-17
Environment: local FastAPI host process, local Supabase Docker stack

## Command

```powershell
uv run --directory apps/api python scripts/perf_probe.py
```

The probe was run with a temporary local API process and a local-only Supabase
access token generated by `apps/api/scripts/local_access_token.py`.

## Interpretation

The shared Supabase HTTP client and safe structured timing logs removed the
previous multi-second local backend behavior for the sampled routes. The next
performance review should focus on frontend request waterfalls and any route
that reappears above the release target under full app usage.
