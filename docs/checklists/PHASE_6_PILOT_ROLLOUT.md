# Phase 6 Checklist — Pilot and Rollout

## Pilot

- [ ] Select pilot users.
- [ ] Import a limited set of real folders.
- [ ] Train pilot users.
- [ ] Observe common mistakes.
- [ ] Collect feedback.
- [ ] Fix blockers.
- [ ] Confirm archive labels work physically.
- [ ] Confirm QR scanning works.
- [ ] Confirm Director Dashboard reflects real activity.

## Production Rollout

- [ ] Freeze release candidate.
- [ ] Create final backup.
- [ ] Deploy production.
- [ ] Run smoke tests.
- [ ] Add approved employees.
- [ ] Roll out department-by-department.
- [ ] Monitor errors.
- [ ] Verify backups after launch.
- [ ] Review audit events.
- [ ] Schedule post-launch review.

## Security Gate

- [ ] Review pilot permissions.
- [ ] Remove temporary access.
- [ ] Confirm no test data leaks into production.
- [ ] Confirm production credentials differ from staging.
- [ ] Confirm restore documentation is accessible to authorized staff.

## Exit Criteria

- [ ] ERP is in controlled production use.
