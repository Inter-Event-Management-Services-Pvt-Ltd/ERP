# Director Dashboard Design

Read:

- `docs/01_PRODUCT_SCOPE.md`
- `docs/14_FRONTEND_DESIGN_SYSTEM.md`
- `docs/checklists/PHASE_4_MANAGEMENT_DIRECTOR.md`
- `docs/checklists/FRONTEND_CLAUDE.md`

Use either Claude Design or Google Stitch where helpful.

Prioritize:

- fast scanning
- critical alerts first
- clear hierarchy
- drill-down actions
- filterable tables
- accessible metrics
- responsive behavior
- restrained motion
- no color-only status communication
- no restricted detail leaks in summary cards

Required areas:

```text
attendance
active projects
upcoming events
overdue tasks
pending approvals
missing documents
checked-out physical files
overdue returns
verification reminders
recent audit activity
```
