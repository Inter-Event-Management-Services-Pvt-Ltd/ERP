# Contributing

## Branching

```text
main        production-ready
staging     integrated pre-production testing
feature/*   scoped feature work
fix/*       scoped bug fixes
```

## Pull Request Requirements

Every PR must include:

- linked phase and checklist
- summary of changes
- database impact
- authorization impact
- tests added or updated
- commands run
- screenshots for UI changes
- migration notes
- rollback notes
- unresolved risks

Use `.github/pull_request_template.md`.
